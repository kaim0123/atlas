## 1. 五大装置

コンピュータは次の**五大装置**の協調で動作します。

| 装置 | 役割 |
|------|------|
| 入力装置 | キーボード、マウスなど。外部からのデータや命令を取り込む。 |
| 出力装置 | ディスプレイ、プリンタなど。結果を外に出す。 |
| 記憶装置 | 主記憶（メモリ）と補助記憶（HDD/SSD）。プログラムとデータを保持。 |
| 演算装置 | 四則演算・論理演算を行う。CPUの一部。 |
| 制御装置 | 各装置への指示の出し方やタイミングを制御。CPUの一部。 |

演算装置と制御装置は**CPU**にまとまっています。主記憶は揮発性、補助記憶は不揮発性です。

## 2. CPUの役割と性能

### 2.1 命令サイクル

CPUは「命令の取り出し（フェッチ）→ 解読（デコード）→ 実行 → 結果の書き戻し」を繰り返します。この一連の流れを**命令サイクル**といい、クロックに合わせて進みます。

### 2.2 性能に関係する用語

- **クロック周波数**: 1秒あたりのクロック数（Hz）。
- **CPI**: 1命令を実行するのに平均何クロックかかるか。
- **MIPS**: 1秒あたり何百万命令を実行できるか。
- **パイプライン**: 複数命令を重ねて処理し、スループットを上げる方式。
- **マルチコア**: 1チップに複数コアを持たせ、並列処理を行う。

## 3. 記憶階層とメモリ

### 3.1 記憶階層

CPUに近いほど高速・小容量、遠いほど低速・大容量です。

```
CPU ← レジスタ ← キャッシュ ← 主記憶（メモリ）← 補助記憶（HDD/SSD）
```

### 3.2 主記憶とキャッシュ

- **DRAM**: 主記憶の主流。集積度が高く大容量化しやすいが、リフレッシュが必要。
- **SRAM**: キャッシュに使う。DRAMより高速だが容量は小さい。
- **キャッシュヒット率**: アクセスしたデータがキャッシュにあった割合。
- **実効アクセス時間**: キャッシュヒット時とミス時をヒット率で加重平均した時間。

### 3.3 補助記憶

- **HDD**: 磁気ディスク。大容量で単価が安いが、シークに時間がかかる。
- **SSD**: 半導体メモリ。ランダムアクセスが速い。
- **光学ディスク**: CD/DVD/BD。リムーバブルで長期保存向き。

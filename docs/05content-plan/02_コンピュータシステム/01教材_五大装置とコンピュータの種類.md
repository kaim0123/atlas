## 1. 五大装置

コンピュータは次の**五大装置**の協調で動作します。

| 装置 | 役割 | 例 |
|------|------|------|
| 入力装置 | 外部からのデータや命令を取り込む | キーボード、マウス |
| 出力装置 | 結果を外に出す | ディスプレイ、プリンタ |
| 記憶装置 | プログラムとデータを保持 | 主記憶（メモリ）、補助記憶（HDD/SSD） |
| 演算装置 | 四則演算・論理演算を行う | CPU の一部 |
| 制御装置 | 各装置への指示とタイミングを制御 | CPU の一部 |

- **演算装置＋制御装置＝CPU**。命令の解釈・実行と、他装置への指示を担う中枢。
- **記憶装置は2種類**: 主記憶は**揮発性**（電源を切ると消える）、補助記憶は**不揮発性**（保持され続ける）。

```
入力 → [ 制御装置 ─ 演算装置 ]（CPU） ⇄ 記憶装置 → 出力
```

矢印はデータと制御信号の流れ。CPU が記憶装置から命令を取り出し、演算し、結果を書き戻す（詳細は [03 CPUの命令実行](03教材_CPUの命令実行.md)）。

## 2. コンピュータの種類

同じ五大装置の構成でも、用途と規模で呼び名が変わります（違いは概要のみ）。

| 種類 | 位置づけ |
|------|----------|
| パーソナルコンピュータ（PC） | 個人向けの汎用機。 |
| ワークステーション | 設計・研究など高性能を要する業務向け。 |
| サーバ | サービスやデータを他の機器へ提供する側。 |
| スマートフォン・タブレット端末 | 携帯性重視。タッチ操作の小型コンピュータ。 |

いずれも「入力・記憶・演算・制御・出力」という枠組みは共通です。

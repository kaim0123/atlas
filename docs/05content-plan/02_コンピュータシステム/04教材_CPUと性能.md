# 04 CPUと性能

クロック・CPI・MIPS など、試験で計算・比較が問われる領域。

> 分割元: `00_分割元_CPUとメモリ.md` §2.2 ／ 計画: [`ディレクトリ.md`](ディレクトリ.md) A-4

---

## 1. 速さを決める指標

| 用語 | 意味 |
|------|------|
| クロック周波数 | 1秒あたりのクロック数（Hz）。大きいほど1拍が速い。 |
| CPI | 1命令の実行に平均何クロックかかるか（Clocks Per Instruction）。 |
| MIPS | 1秒あたり何百万命令を実行できるか。 |

**関係**: クロック周波数が高く、CPI が小さいほど、単位時間に多くの命令を処理できる。

```
1命令あたりの時間 = CPI ÷ クロック周波数
MIPS = クロック周波数(MHz) ÷ CPI
```

**例**: 2 GHz・CPI 2 なら、1命令 = 2 ÷ 2×10⁹ = 1 ns、MIPS = 2000 ÷ 2 = 1000 MIPS。

---

## 2. 高速化のしくみ

- **パイプライン**: 複数命令のフェッチ・デコード・実行を少しずつずらして重ね、スループットを上げる。分岐で乱れると効果が下がる。
- **マルチコア**: 1チップに複数コアを載せ、複数の処理を同時に走らせる。

> より深い並列処理の分類（SISD/MIMD・アムダールの法則）は発展として [05 並列処理](05教材_並列処理.md) に分離。
